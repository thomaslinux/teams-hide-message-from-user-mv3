manifest v3 extension based on this working userscript :

const hideUsers = ["USER1", "USER2"];

window.addEventListener("load", () => {
  console.log("Hide users:", hideUsers);


  // Build the CSS selector from hideUsers
  const selector = hideUsers
    .map(
      (u) =>
        `div[class*="fui-ChatMessage"]:has(img[src*="${u}"]) div[data-message-content]`,
    )
    .join(",\n");


create a <style id="tl9hideUsers"></style>
create a <style id="tl9bgUrl></style>

options :
- popup option on click on the extension icon
<h3> Users to hide
- add / remove a user to the list of users to hide (Enter to add a user to the list)
- display the users in a table th FAMILYNAME Hide Remove
- enable disable hiding of that user (checkbox) 
- multiple users can be hidden
- radio between
  - hide message content only
  - hide message. hide message uses this line :
         `div[class*="fui-ChatMessage"]:has(img[src*="${u}"])`,
  - show only those messages. This uses this style :
       div[class*="fui-ChatMessage"]:has(img) div[data-message-content] {
        display: none;
      }
      div[class*="fui-ChatMessage"]:has(img[src*="LASTNAME"]) div[data-message-content],
          display: inherit;
      } 
<h3> My messages
- radio toggle between
  - Normal (show all)
  - Hide all my messages
        div[class*="fui-ChatMyMessage"] {
        display:none;
    }
  - Show only my messages

<h3> Background Image
- add / remove a background URL (Enter to add)
- display the URLs in a table th USE BackgroundURL Remove (radio selector)
